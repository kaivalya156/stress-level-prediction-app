# Stress Level Prediction App

This project predicts student stress levels based on lifestyle and behavioral factors using Machine Learning.

## Features
- Interactive web app built using Streamlit
- Predicts stress based on mood, sleep, screen time, and social media usage
- Uses trained ML model with feature engineering
- Includes rule-based adjustments for realistic predictions
- Visualizes input data

## Technologies Used
- Python
- pandas, numpy
- scikit-learn
- Streamlit

## How to Run
1. Install dependencies:
   pip install streamlit pandas numpy scikit-learn

2. Run the app:
   streamlit run app.py
